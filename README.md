# myuw-boilerplate
Boilerplate sandbox using Jekyll.

## Install Jekyll on OS X (if you haven't already)

    $ gem install jekyll bundler
    
## Clone this repo

    $ git clone https://github.com/charlon/myuw-boilerplate.git
    $ cd myuw-boilerplate

And then execute:

    $ bundle

## Jekyll Commands

    $ jekyll serve
